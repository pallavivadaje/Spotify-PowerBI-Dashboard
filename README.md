# 🎵 Spotify Power BI Dashboard

## 📌 Project Overview

This project is an interactive **Power BI Dashboard** built using Spotify music data. It provides insights into songs, artists, albums, popularity, and music trends through visually appealing dashboards.

The dashboard helps users explore music performance using interactive charts, KPIs, filters, and drill-down analysis.



## 📊 Dashboard Highlights

- 🎵 Total Songs
- 🎤 Total Artists
- ⭐ Average Song Popularity
- ⏱ Average Song Duration
- 📅 Songs Released by Year
- 📆 Monthly Song Analysis
- 💿 Songs by Album Type
- 🔥 Popular Songs
- 🎧 Popular Artists
- 📈 Popularity Trends
- 📋 Detailed Song Information Table


## 🛠 Tools Used

- Power BI Desktop
- Microsoft Excel / CSV Dataset
- Power Query
- DAX (Data Analysis Expressions)
